# SmartSpend – Expense Tracker (Spring Boot + Thymeleaf + MySQL)

SmartSpend is a personal finance web application to track **expenses & income**, set **monthly budgets**, receive **budget alerts/notifications**, and export reports to **PDF/Excel**.  
It also includes **Email OTP verification** for registration and **Forgot Password** reset links.

---

## Features

- ✅ User Registration + Email OTP Verification  
- ✅ Login/Logout (Session-based)
- ✅ Add / Edit / Delete Expenses
- ✅ Filter & Sort expenses (type, category, date range, amount range)
- ✅ Set Monthly Budgets (per category)
- ✅ Budget alerts (Near limit / Exceeded)
  - In-app notifications
  - Email alerts
- ✅ Export Expense Report:
  - PDF
  - Excel
- ✅ Forgot Password (Email reset link)

---

## Tech Stack

- **Java 21**
- **Spring Boot 4**
- **Spring MVC + Thymeleaf**
- **Spring Data JPA (Hibernate)**
- **MySQL**
- **Bootstrap 5**
- **SendGrid (Email)** *(recommended for Railway deployment)*
- **WebSocket / STOMP** (real-time notifications)
